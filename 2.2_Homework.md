* 统计1.gtf列数
``` bash
> awk 'BEGIN{max=0}NF>max{max=NF}END{print max}' 1.gtf
34
```
以默认的空格和制表符计算，最大列数为34列；
``` bash
> awk -F"\t" 'BEGIN{max=0}NF>max{max=NF}END{print max}' 1.gtf
9
```
如指定仅以制表符分列，则共有9列
